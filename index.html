  1. Создайте форму регистрации (логин, пароль, e-mail)

Урок 6. Работа с базой данных
  1. Создайте таблицу с пользователями (id, имя пользователя, путь к картинке с его фотографией)
  2. Добавьте в неё 5-7 записей
  3. Выведите на отдельной странице (действите «users») сайта список пользователей из базы данных (в списке должны присутствовать id, имена пользователей и их фотографии)
Урок 7. Создание навигации по страницам
  1. Создайте Pagination для страницы с пользователями, чтобы на одной странице выводились только 2 человека
Урок 8. Работа с URL
  1. Создайте и выведите несколько произвольных URL на отдельную страницу (в теге )
  2. Исключите index.php из адресной строки
  3. Добейтесь замены ссылки вида http://mysite.local/web/?r=site/users на http://mysite.local/web/users
Урок 9. Обработка запросов
  1. Добавьте в адресную строку GET-параметр с id пользователя. Должно получиться что-то наподобие этого: http://mysite.local/web/user?id=5
  2. Выведите на этой странице того пользователя, чей id был указан. Если id выбран несуществующий, то выведите «Указанного пользователя не существует»
Урок 10. Работа с сессиями
  1. Добавьте 2 ссылки на главную страницу сайта с выбором размера шрифта (2 ссылки: «крупный», «нормальный»)
  2. Запишите выбор пользователя в сессию после перехода по ссылке
  3. При заходе на страницу с таблицей с пользователями сделайте текст в ней согласно выбранному ранее размеру
Урок 11. Работа с cookie
  1. Переделайте предыдущее задание, но используя уже cookie
Урок 12. Создание виджетов
  1. Создайте виджет, выводящий простое меню в виде:
    <ul>
      <li>
        <a href=«Ссылка 1»>Пункт 1</a>
      </li>
      <li>
        <a href=«Ссылка 2»>Пункт 2</a>
      </li>
    </ul>
  2. Выведите его на странице с пользователями с несколькими пунктами.
